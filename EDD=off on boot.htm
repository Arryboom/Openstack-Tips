    EDD has been around for a long time, not something new - quite old in fact. It supplies additional support for IDE drives via the BIOS Int 13h interface allowing larger disks, more function such as more than two disks, more DMA modes, CHS -independent methods for accessing the drives, etc.

    A comprehensive explanation here... http://www-pc.uni-regensburg.de/hardware/TECHDOK/ATA_EDD_11.PDF

    I tried to warn you off that board :-) EDD is "BIOS Enhanced Disk Drive Services"
